# Project Brief — DAKSA MVP

## 📋 Visão Geral

**DAKSA** é um SaaS para gestão de clínicas de saúde. MVP focado em:
- Agendamentos com automação WhatsApp
- Gestão de clientes + anamnesia versionada
- Protocolos de tratamento + analytics

**Alvo:** Terapeutas, psicólogos, fisioterapeutas com 1-20 profissionais.

## 🎯 Objetivos (Mês 1-3)

| Objetivo | Métrica | Target |
|----------|---------|--------|
| MVP funcional | Deployment em produção | Semana 7 |
| Pronto para usuários | 5+ clínicas usando | Mês 2 |
| Retention | % ativo após 30 dias | >70% |
| NPS | Net Promoter Score | >40 |
| Performance | p95 latência | <500ms |

## 👥 Personas

### Persona 1: Terapeuta Independente
- **Nome:** Bruna, 35 anos
- **Pain points:** Agenda desorganizada, sem confirmação, sem histórico
- **Necessidades:** Agenda visual, confirmação automática, anamnesia estruturada

### Persona 2: Gerente de Clínica
- **Nome:** Carlos, 40 anos
- **Context:** 5 terapeutas, 100+ clientes
- **Pain points:** Múltiplas agendas, sem visibilidade, sem dados
- **Necessidades:** Dashboard, alertas, métricas

## 📊 Modelo de Negócio

### Pricing (MVP Simplificado)
- **Free Plan:** 1 terapeuta, 20 clientes, 30 agendamentos/mês
- **Pro Plan:** $99/mês, ilimitado, analytics, suporte
- **Enterprise:** Custom, SLA 99.9%

**MVP:** Apenas Free + plano manual

## 📈 Fases

### Phase 1: MVP Core (Semana 1-7)
- Agenda, Clientes, Protocolos, WhatsApp, Auth

### Phase 2: Refinement (Mês 2)
- Stripe, Email, Dark mode, Google Calendar, Accessibility

### Phase 3: Growth (Mês 3+)
- Portal cliente, API pública, Marketplace, Integrações
