# Tech Stack — DAKSA MVP

## Stack Detalhado

**Frontend:** Next.js 15 + TypeScript + Tailwind CSS  
**Backend:** Fastify + Prisma 5  
**Database:** PostgreSQL 16 + Redis 7  
**Auth:** NextAuth.js + JWT  
**Infra:** Docker Compose + Traefik  
**Testing:** Jest + Cucumber  

## Justificativas Principais

- **Next.js:** SSR, file-based routing, Vercel-ready
- **Fastify:** 2x Express, JSON Schema nativo
- **Prisma:** Type-safe, migrations auto, DX excelente
- **PostgreSQL:** Robusto, EXCLUDE constraint (overlap detection), JSONB
- **Docker:** Replicável (dev == prod)
