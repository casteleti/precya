# PRECY — DESIGN SYSTEM & UI/UX GUIDELINES v1.0

## Overview

Precy is not a traditional ERP or cold SaaS dashboard.

It is an emotionally intelligent financial and operational platform designed primarily for women entrepreneurs in the beauty and aesthetics industry.

The product must feel:

* calming
* sophisticated
* supportive
* elegant
* emotionally safe
* professional without intimidation

The interface should reduce anxiety, cognitive overload and emotional resistance related to pricing, finances and clinic management.

The experience should feel closer to:

* a wellness premium platform
* a caring financial assistant
* a calm operational companion

NOT:

* fintech aggressive UI
* enterprise ERP
* startup neon dashboard
* productivity obsession interface

---

# CORE EMOTIONAL PRINCIPLES

## Emotional Positioning

The UI should communicate:

> "You are capable. Your work has value. We help you organize and grow calmly."

The platform should never:

* shame the user
* create pressure
* feel noisy
* look overly technical
* overwhelm visually

# TYPOGRAPHY

## Primary Font

### Recommended

* Inter
* Poppins

## Style Rules

### Titles

* Weight: 500
* Size: 24px–32px
* Line-height: 1.2
* Spacious margins

### Body Text

* Size: 16px–18px
* Weight: 400
* Line-height: 1.6

### Buttons

* Weight: 500
* Avoid fully bold buttons
* Sentence case preferred

---

# BORDER RADIUS

## Global Radius

```css
--radius-sm: 8px;
--radius-md: 12px;
--radius-lg: 16px;
--radius-xl: 24px;
```

Rounded corners are essential.

Sharp edges should be avoided.

---

# SHADOW SYSTEM

All shadows must be:

* soft
* diffused
* elegant

Never harsh.

```css
--shadow-soft:
0 4px 20px rgba(0,0,0,0.06);

--shadow-card:
0 8px 32px rgba(0,0,0,0.04);
```
